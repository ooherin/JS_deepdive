타이머 함수 setInterval, setTimeout은 일정 시간이 경과된 이후 콜백함수가 호출되도록
타이머를 생성한다. setTimeout은 한번만 동작하고, setInterval는 타이머가 
만료될때마다 반복 호출한다. 
이 함수는 clearTimeout, clearInterval로 제거할 수  있다.

js엔진은 싱글 스레드로 동작하나, 이 타이머 함수들은 비동기 처리 방식으로 동작한다. 

디바운스 : 짧은 시간 간격으로 이벤트가 연속으로 발생하면 이벤트 핸들러를 호출하지 않다가
일정시간이 경과한 이후에 이벤트 핸들러가 한 번만 호출되도록 하는 기능
-> resize 이벤트 처리나 input 요소에 입략된 값으로 ajax를 요청하는 입력 필드 자동완성
UI 구현, 버튼 중복 클릭 방지 처리 등에 유용하게 사용됨

스로틀 : 짧은 시간 간격으로 이벤트가 연속해서 발생하더라도 일정 시간 간격으로 이벤트 핸들러가 최대 한번만 호출
되도록 하는 ㅎ기능. 짧은 시간 간격으로 연속해서 발생하는 이벤트를 그룹화해 
일정 시간 단위로 이벤트 핸들러가  호출되도록 호출 주기를 만듦.

스크롤할때 이벤트는 연속해서 발생하는데, 스로틀은 두번째 인수로 전달한 시간(delay) 가 경과하기 이전에 이벤트가 발생하면 아무것도 하지 않다가 경과
하면 콜백함수를 호출하고 새로운 타이머를 재설정한다.
scroll이벤트 처리나 무한 스크롤 UI 구현에 사용된다.
