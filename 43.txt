Ajax란 js를 사용하여 브라우저가 서버에게 비동기 방식으로 데이터를 요청하고,
서버가 응답한 데이터를 수신하여 웹페이지를 동적으로 갱신하는 프로그래밍 방식을 
말한다.
Ajax는 브라우저에서 제공하는 Web APi인 XMLHttpRequest 객체를 기반으로 동작한다. 
XMLHttpRequest는 HTTP 비도기 통신을 위한 메서드, 프로퍼티를 제공한다. 

Ajax이전에는 웹페이지 전체를 처음부터 다시 렌더링하는 방식으로 동작했다.
화면이 전환되면 변하지 않는 부분까지도 모두 재렌더링하기 때문에 깜박임 현상, 불필요한 
데이터 통신, 블로킹 등의 문제가 있었다.
Ajax는 웹페이지 변경에 필요한 데이터만 비동기 방식으로 전송받아 일부만 재렌더링 함으로써
부드러운 화면 전환이 가능하다. 

다음과 같은 장점이 있다.
1. 변경할 부분을 갱신하는 데 필요한 데이터만 받아 최적의 데이터 통신
2. 필요없는 부분은 렌더링 안함. 깜박임 현상 없음
3. 클라이언트와 서버와의 통신이 비동기 방식으로 동작하므로 서버에게 요청을
보낸 후 블로킹이 발생하지 않음

JSON
JSON은 클라이언트와 서버 간의 HTTP통신을 위한 텍스트 테이터 포맷이다. 
JSON의 키는 반드시 큰따옴표로 묶어야 한다. 객체 값의 경우 문자열은 ""(큰따옴표)사용해야 한다. 

JSON의 표기방식
{
    "name" : "Lee",
    "age" :  20,
    "alive" :true,
    "hobby" : ["traveling","tennis"]
}
